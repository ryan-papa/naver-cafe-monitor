**ID:** 20260411_120000_naver-cafe-monitor_b7e2c9f4
**날짜:** 2026-04-11
**상태:** Approved

---

## 개요·목적

- **배경:** 네이버 카페 특정 게시판을 수동으로 반복 확인하는 번거로움을 자동화
- **목표:** 새 게시물 자동 감지 → 게시판 유형별 처리(이미지 필터링 / 텍스트 요약) → 카카오 오픈채팅 봇 알림 전달
- **범위:** 지정 카페·게시판 크롤링, 자녀 이미지 필터, 공지 AI 요약, 카카오 봇 메시징, 폴링 스케줄러

---

## 사용자 스토리

| As a | I want | So that |
|------|--------|---------|
| 부모 사용자 | 자녀가 포함된 카페 이미지를 자동 수신 | 카페를 직접 반복 확인하지 않아도 됨 |
| 부모 사용자 | 공지사항 일정을 AI 요약본으로 수신 | 중요 일정을 놓치지 않음 |
| 운영자 | 폴링 주기 설정 및 ON/OFF 토글 | 상황에 따라 알림 빈도 조절 가능 |

| F-13 | 이미지 판독 불가 시 처리 정책 적용 (OI-01 해소 후 구현) | Low |

## 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 실행 환경 | 로컬 우선, Docker 컨테이너화로 클라우드 이관 가능한 구조 |
| 성능 | 폴링 1회 처리 시간 < 설정 주기 (데드라인 초과 시 경고 로그) |

| 신뢰성 | 크롤링 실패 시 최대 N회 재시도 + 오류 로그 기록 (재시도 횟수·간격 config 설정) |
| 데이터 보존 | 다운로드 이미지 로컬 저장 여부·보관 기간 정책 (OI-02 해소 후 확정) |
