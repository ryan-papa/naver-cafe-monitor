**날짜:** 2026-04-11
**상태:** Approved

## 개요·목적

- **배경:** 네이버 카페의 특정 게시판을 수동으로 확인하는 번거로움을 자동화
- **목표:** 새 게시물 자동 감지 → 유형별 처리(이미지 필터링 / 텍스트 요약) → 카카오 오픈채팅 봇으로 알림 전달
- **범위:** 지정 카페·게시판 크롤링, 이미지 얼굴 인식 필터, 공지 AI 요약, 카카오 봇 메시징

---

## 사용자 스토리

| As a | I want | So that |
|------|--------|---------|
| 부모 사용자 | 자녀가 포함된 카페 이미지를 자동으로 수신 | 직접 카페를 반복 확인하지 않아도 됨 |
| 부모 사용자 | 공지사항 일정을 AI 요약본으로 수신 | 중요 일정을 놓치지 않음 |
| 운영자 | 폴링 주기를 설정하고 ON/OFF 토글 | 상황에 따라 알림 빈도 조절 |

## 제약사항

- **기술:**
  - 네이버 카페 공식 API 없음 → Selenium 또는 Playwright 기반 크롤링 사용
  - 카카오톡 공식 개인 전송 API 없음 → 오픈채팅 봇 비공식 방식 사용
