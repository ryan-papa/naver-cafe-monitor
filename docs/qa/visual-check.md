# Visual Check — naver-cafe-monitor 전면 리디자인

| 항목 | 내용 |
|------|------|
| 대상 | 관리자 콘솔 전면 UI 리디자인 (museum-finder 토큰 이식) |
| 실행 주체 | QA 담당자 (수동) |
| 실행 시점 | 각 배포 직전 + 배포 직후 1회 |
| 통과 기준 | 모든 항목 체크 + 스크린샷 아카이브 |

## Shell·브랜드

- [ ] `admin` 상단 좌측 **NC 로고** 22×22, 4px 라운드, 다크 배경 + 흰 글자 `NC`
- [ ] 로고 우측 **"Naver Cafe Monitor"** (fs-13, weight 600) + **"Admin · v0.0.1"** (fs-11, fg-faint)
- [ ] 사이드바 섹션 헤더 `개요` / `모니터링` / `시스템` 구분 및 간격 일관

## 게시물·처리 이력

- [ ] 테이블 헤더 `ID / 게시판 / 제목 / 작성자 / 작성일 / 상태 / 액션` fs-12, fg-faint
- [ ] 행 hover 시 `bg-hover` 배경, 활성 행은 `bg-selected`
- [ ] 공개 배지 Pill `tone=success` — `--success-soft` 배경 + `--success-fg` 글자
- [ ] 비공개 배지 Pill `tone=muted`
- [ ] 각 행 우측 **액션 버튼** 세트 (상세 · 재발송) 균일 정렬

## 필터·페이지네이션

- [ ] 필터 칩 셀렉트 (카페 / 게시판 / 상태 / 기간) 스타일 일관, 레이블은 좌측 상단
- [ ] 페이지네이션 `1-10 / 총합 N` 텍스트 좌측, **"10개씩"** 셀렉트 + `≪ < > ≫` 4버튼 우측
- [ ] 빈 상태 empty-state 일러스트 또는 안내 문구 표시

## 모달·재발송 플로우

- [ ] 상세 모달 열림 시 배경 `rgba(0,0,0,0.4)` + Esc·오버레이 클릭으로 닫힘
- [ ] 모달 오픈 시 포커스가 첫 번째 포커서블 요소로 이동 (focus-trap)
- [ ] **재발송 버튼** 클릭 시 30초 카운트다운 + 버튼 비활성화, 0초에 자동 재활성

## 인증 페이지

- [ ] `/login` 카드 max-width 400px 중앙 배치, 모바일(<480px)은 shadow 제거
- [ ] TOTP 스텝 최초 숨김, `totp_required` 응답 시 애니메이션 없이 표시
- [ ] 네트워크 오류·invalid_credentials 등 에러 메시지 한국어 표시

## 에러 페이지

- [ ] `/error/404` · `/error/500` · `/error/401` · `/error/403` · `/error/offline` 각 페이지 아이콘·제목·설명·CTA 일관
- [ ] **"대시보드로 돌아가기"** 링크 정상 동작 (미인증 시 /login 리다이렉트 포함)

## 타이포·폰트

- [ ] NanumSquare 본문 폰트 로드 (FOUC 미발생, 첫 렌더부터 적용)
- [ ] 숫자 영역은 `font-mono` (ID·버전·타임스탬프) — 로고와 이력 테이블 ID 컬럼
- [ ] 헤딩 스케일 fs-11 / fs-12 / fs-13 / fs-18 토큰 일관 적용

## 접근성

- [ ] Tab 순서가 논리적 (좌상단 → 메인 콘텐츠 → 우측 액션 → 사이드바)
- [ ] 포커스 링이 모든 인터랙션 요소에 **가시적**으로 표시
- [ ] 색 대비: 본문 fg 대 bg 4.5:1, 보조 fg-muted 대 bg 3.0:1 이상
- [ ] 스크린리더로 `/login` · `/admin` 초기 렌더링 시 안내 순서 자연스러움
